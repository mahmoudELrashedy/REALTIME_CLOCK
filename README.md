#smart_watch

in this project I used atmega32 MCU to implement a smartwatch
with different modes\
-clock mode\
-alarm mode\
-stopwatch mode

HAL :\
-LCD \
-seven segment \
-keypad

MCAL :\
-DIO\
-TMR0 CTC\
-EXTI(extrnal interrupt)\
